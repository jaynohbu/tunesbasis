Sequence Repetitions with *

In Python, using * with sequences (like lists, strings, or tuples) and an integer repeats the sequence multiple times. With two integers, it’s just multiplication.

Examples

0 * 4       # 0                 (int × int)
[0] * 4     # [0, 0, 0, 0]      (list × int)
"ab" * 3    # "ababab"          (str × int)
(1, 2) * 2  # (1, 2, 1, 2)      (tuple × int)

Note

• Works with list, str, tuple.
• int × int behaves as arithmetic multiplication.
• Be careful: [[0]] * 4 makes 4 references to the same list — modifying one modifies all.
